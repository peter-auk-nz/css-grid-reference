CSS Reference Guide

A quick reference for CSS fundamentals

Selectors

  • Type selector - Example: p { ... }
  • Class - Example: .box { ... }
  • ID - Example: #header { ... }
  • Attribute - Example: [type="text"]

Pseudo Classes

  • Hover - Example: a:hover { ... }
  • Focus - Example: input:focus { ... }
  • First-child - Example: li:first-child { ... }
  • Nth-child - Example: li:nth-child(2) { ... }

Grid Layout

  • Container - Example: display: grid;
  • Columns - Example: grid-template-columns: 1fr 1fr;
  • Gap - Example: gap: 20px;
  • Areas - Example: grid-template-areas: "header";

Flexible Box Layout

  • Container - Example: display: flex;
  • Direction - Example: flex-direction: row;
  • Justify - Example: justify-content: center;
  • Align - Example: align-items: center;

Media Queries

  • Mobile - Example: @media (max-width: 768px)
  • Desktop - Example: @media (min-width: 1024px)
  • Print - Example: @media print
  • Dark mode - Example: @media (prefers-color-scheme: dark)

Color

  • Named - Example: color: red;
  • Hex - Example: color: #3498db;
  • RGB - Example: color: rgb(255, 0, 0);
  • HSL - Example: color: hsl(120, 100%, 50%);
