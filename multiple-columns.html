Multiple Columns

Create newspaper-style multi-column layouts for text content

What are Multiple Columns?

CSS multiple columns let you flow text into multiple vertical columns, like a newspaper or magazine. This improves readability for long text passages and creates professional-looking layouts without complex grid or flexbox code.

Key Use: Perfect for articles, blog posts, and text-heavy content.

column-count

Specifies the number of columns.

.text { column-count: 3; }

Example: Divides content into 3 equal-width columns

Auto: Browser determines count based on column-width

column-width

Suggests an optimal width for columns.

.text { column-width: 200px; }

Not exact: Browser creates as many ~200px columns as will fit

Responsive: Column count adjusts automatically to viewport

Best practice: Often better than column-count for responsiveness

columns (Shorthand)

Combines column-width and column-count.

.text { columns: 200px 3; }

Syntax: width count (either order)

Example: Create 3 columns, each about 200px wide

column-gap

Sets the space between columns.

.text { column-gap: 2rem; }

Default: Usually 1em

Example: Creates 2rem gutter between columns

column-rule

Adds a dividing line between columns.

.text { column-rule: 1px solid #ccc; }

Syntax: width style color (like border)

Styles: solid, dotted, dashed, double

Note: Doesn't take up space (unlike borders)

column-span

Makes an element span across all columns.

h2 { column-span: all; }

Values: none (default), all

Use: Headers, images, pull quotes that should span full width

column-fill

Controls how content is distributed across columns.

.text { column-fill: balance; }

balance (default): Distribute content evenly across columns

auto: Fill columns sequentially

break-inside

Prevents elements from breaking across columns.

img { break-inside: avoid; }

Use: Keep images, code blocks, or elements together in one column

Values: auto, avoid, avoid-page, avoid-column
