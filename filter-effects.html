Filter Effects

Apply graphical effects like blur, brightness, and contrast to elements

← Back to Reference

What are Filter Effects?

CSS filters apply graphical effects to elements, similar to photo editing software. They can blur, adjust colors, add shadows, and transform appearance without needing image editing tools. Filters are hardware-accelerated and work on any element, including images, backgrounds, and text.

Key Advantage: Real-time visual effects without external image editors.

blur()

Applies a Gaussian blur to the element.

filter: blur(5px);

Values: Length units (px, rem, em)

Example: blur(5px) creates moderate blur effect

Use: Backgrounds, focus effects, loading states

brightness()

Adjusts the brightness of the element.

filter: brightness(150%);

Values: Percentage or decimal (0% = black, 100% = normal, 200% = double bright)

Example: brightness(50%) darkens by half, brightness(200%) doubles brightness

Use: Darken/lighten images, hover effects

contrast()

Adjusts the contrast of the element.

filter: contrast(200%);

Values: Percentage or decimal (0% = gray, 100% = normal, 200+ = high contrast)

Example: contrast(150%) increases difference between light and dark

Use: Enhance images, create dramatic effects

grayscale()

Converts element to grayscale (black and white).

filter: grayscale(100%);

Values: 0% to 100% (0% = color, 100% = full grayscale)

Example: grayscale(50%) creates partial desaturation

Use: Hover effects (color on hover), vintage looks, disabled states

saturate()

Adjusts color saturation (intensity).

filter: saturate(200%);

Values: Percentage or decimal (0% = grayscale, 100% = normal, 200+ = oversaturated)

Example: saturate(150%) makes colors more vibrant

Use: Enhance photos, create vivid effects

hue-rotate()

Rotates colors around the color wheel.

filter: hue-rotate(90deg);

Values: Degrees (0deg to 360deg)

Example: hue-rotate(180deg) inverts colors to opposite on color wheel

Use: Color variations, theme switching, creative effects

invert()

Inverts the colors of the element.

filter: invert(100%);

Values: 0% to 100% (0% = normal, 100% = fully inverted)

Example: invert(100%) creates negative image effect

Use: Dark mode, creative effects, icons

sepia()

Applies a sepia tone (warm brownish tint).

filter: sepia(100%);

Values: 0% to 100% (0% = normal, 100% = full sepia)

Use: Vintage effects, old photo look

drop-shadow()

Adds a drop shadow following the element's shape.

filter: drop-shadow(5px 5px 10px rgba(0,0,0,0.5));

Syntax: offset-x offset-y blur-radius color

vs box-shadow: Follows element outline, not box

Use: Shadows on PNG images, irregular shapes

Multiple Filters

Combine multiple filters for complex effects.

filter: grayscale(100%) contrast(120%) brightness(90%);

Order: Filters apply left to right

Example: Create vintage high-contrast black and white effect

backdrop-filter

Applies filters to the area behind an element.

backdrop-filter: blur(10px);

Use: Frosted glass effects, blurred backgrounds behind modals

Note: Same filter functions as regular filter property
