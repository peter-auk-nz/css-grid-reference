CSS Grid Layout

Two-dimensional layout system for creating complex responsive designs

What is Grid Layout?

CSS Grid is a powerful two-dimensional layout system that lets you arrange elements in rows and columns simultaneously. It's perfect for page layouts, image galleries, and complex designs.

Key Advantage: Control both rows and columns at the same time, unlike Flexbox which is one-dimensional.

Creating a Grid Container

Turn any element into a grid container to control its children.

.container { display: grid; }

Effect: Direct children become grid items

grid-template-columns

Defines the column structure of your grid.

.container { grid-template-columns: 200px 200px 200px; }

Example: Creates 3 columns, each 200px wide

Using fr units: grid-template-columns: 1fr 1fr 1fr; (3 equal flexible columns)

Using repeat(): grid-template-columns: repeat(3, 1fr); (shorthand for 3 equal columns)

grid-template-rows

Defines the row structure of your grid.

.container { grid-template-rows: 100px auto 100px; }

Example: First row 100px, middle row auto-sized, last row 100px

gap (grid-gap)

Creates space between grid items.

.container { gap: 20px; }

Example: 20px space between all rows and columns

Separate control: row-gap: 20px; column-gap: 10px;

Placing Items

Control where individual items appear in the grid.

.item { grid-column: 1 / 3; grid-row: 1 / 2; }

Example: Item spans from column line 1 to 3 (2 columns wide), in row 1

Shorthand: grid-area: row-start / column-start / row-end / column-end;
