# CSS Reference

A comprehensive CSS reference guide built with CSS Grid layout, featuring 18 essential categories covering everything from selectors to advanced visual effects.

## 🌐 Live Site

**[View Live Site](https://peter-auk-nz.github.io/css-reference/)**

## 📋 Features

This reference guide covers 18 CSS categories:

- **Selectors** - Target HTML elements effectively
- **Pseudo Classes** - Style element states and positions
- **Grid Layout** - Modern two-dimensional layouts
- **Flexible Box Layout** - One-dimensional flexible layouts
- **Media Queries** - Responsive design breakpoints
- **Color** - Color values and formats
- **2D Transforms** - Rotate, scale, translate, and skew
- **3D Transforms** - Three-dimensional transformations
- **Animation** - Keyframe animations
- **Typographic Styles** - Font properties and text styling
- **Text Effects** - Shadows, transforms, and decorations
- **Gradients** - Linear, radial, and conic gradients
- **Values and Sizing** - Units and measurements
- **Multiple Columns** - Multi-column text layouts
- **Opacity** - Transparency controls
- **Filter Effects** - Visual filters and adjustments
- **Blend Modes** - Color and layer blending
- **Masking** - Clipping and masking techniques

## 📖 About This Project

This CSS reference guide started as a concept in 2015 and was completed in 2026. It demonstrates modern CSS Grid layout techniques while serving as a practical quick-reference tool for web developers.

The site uses a clean 3×6 grid layout that adapts responsively to different screen sizes, making it accessible on any device.
