Blend Modes

Control how elements blend with their backgrounds and each other

← Back to Reference

What are Blend Modes?

Blend modes determine how an element's colors mix with the colors behind it. They work similarly to blending modes in photo editing software like Photoshop. By changing the blend mode, you can create dramatic visual effects, overlays, and color adjustments without editing images.

Key Concept: Blend modes affect how layers interact visually.

mix-blend-mode

Controls how an element blends with elements behind it.

.overlay { mix-blend-mode: multiply; }

Applies to: The entire element and its background

Use: Image overlays, text effects, creative compositions

background-blend-mode

Controls how multiple background images blend together.

background-blend-mode: multiply;

Use: Combine background images, create duotones, tint images

Works with: Multiple background-image layers

Common Blend Modes

Most frequently used blending effects.

mix-blend-mode: normal; /* Default, no blending */
mix-blend-mode: multiply; /* Darkens */
mix-blend-mode: screen; /* Lightens */
mix-blend-mode: overlay; /* Combines multiply and screen */

multiply: Colors become darker, white disappears

screen: Colors become lighter, black disappears

overlay: Increases contrast, preserves highlights and shadows

Darken Modes

Blend modes that create darker results.

mix-blend-mode: darken;
mix-blend-mode: multiply;
mix-blend-mode: color-burn;

darken: Keeps darkest color from each layer

multiply: Multiplies colors (always darker unless white)

color-burn: Increases contrast, creates dramatic darks

Lighten Modes

Blend modes that create lighter results.

mix-blend-mode: lighten;
mix-blend-mode: screen;
mix-blend-mode: color-dodge;

lighten: Keeps lightest color from each layer

screen: Inverse of multiply (always lighter unless black)

color-dodge: Brightens colors, creates glowing effects

Contrast Modes

Blend modes that adjust contrast.

mix-blend-mode: overlay;
mix-blend-mode: soft-light;
mix-blend-mode: hard-light;

overlay: Multiply dark areas, screen light areas

soft-light: Subtle version of overlay

hard-light: Strong version of overlay

Color Modes

Blend modes that affect color separately from brightness.

mix-blend-mode: hue;
mix-blend-mode: saturation;
mix-blend-mode: color;
mix-blend-mode: luminosity;

hue: Uses hue from top, saturation/luminosity from bottom

saturation: Uses saturation from top, hue/luminosity from bottom

color: Uses hue and saturation from top, luminosity from bottom

luminosity: Uses luminosity from top, hue/saturation from bottom

Difference Modes

Create inverted or unique color effects.

mix-blend-mode: difference;
mix-blend-mode: exclusion;

difference: Subtracts colors, creates high contrast

exclusion: Similar to difference but lower contrast

Use: Creative effects, unique color combinations

Practical Examples

Common use cases for blend modes.

/* Duotone image effect */
background: url(image.jpg), linear-gradient(blue, red);
background-blend-mode: multiply;

/* Text over image */
h1 { mix-blend-mode: screen; color: white; }

Use cases: Photo tints, text effects, creative overlays, color grading
