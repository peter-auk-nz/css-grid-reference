Masking

Create complex shapes and hide portions of elements using masks

What is CSS Masking?

CSS masking lets you hide portions of an element using shapes, gradients, or images. Unlike clipping which creates hard edges, masks can have soft transitions and complex patterns. Masks are perfect for creating non-rectangular shapes, fade effects, and artistic compositions.

Key Concept: Black in the mask = visible, white = hidden, gray = partially transparent.

mask-image

Defines the mask using an image, gradient, or SVG.

mask-image: url(mask.png);
mask-image: linear-gradient(black, transparent);

Image masks: Use alpha channel or luminance to determine visibility

Gradient masks: Create fade effects and transitions

Note: Black = visible, white/transparent = hidden

mask-size

Controls the size of the mask.

mask-size: cover;
mask-size: 100px 200px;

Values: cover, contain, auto, or specific dimensions

Like background-size: Same sizing options

mask-position

Positions the mask within the element.

mask-position: center;
mask-position: top right;

Keywords: top, bottom, left, right, center

Values: Percentages or lengths (50% 50%, 10px 20px)

mask-repeat

Controls whether the mask repeats.

mask-repeat: no-repeat;
mask-repeat: repeat;

Values: repeat, no-repeat, repeat-x, repeat-y, space, round

Like background-repeat: Same repetition options

mask-mode

Determines how the mask is calculated.

mask-mode: alpha;
mask-mode: luminance;

alpha: Uses transparency channel

luminance: Uses brightness of colors (black/white)

Default: Usually alpha for PNG images

mask Shorthand

Combines multiple mask properties.

mask: url(mask.png) center/cover no-repeat;

Syntax: image position/size repeat

Like background shorthand: Same pattern

Gradient Masks

Create fade effects using gradients as masks.

/* Fade from top */
mask-image: linear-gradient(to bottom, black, transparent);

/* Circular fade */
mask-image: radial-gradient(circle, black 50%, transparent);

Use: Image fades, vignettes, spotlight effects

Multiple Masks

Combine multiple masks for complex effects.

mask-image: url(mask1.png), url(mask2.png);
mask-composite: add;

mask-composite: Determines how masks combine

Values: add, subtract, intersect, exclude

clip-path vs mask

Understanding the difference between clipping and masking.

/* Hard edges */
clip-path: circle(50%);

/* Soft edges possible */
mask-image: radial-gradient(circle, black, transparent);

clip-path: Hard edges, geometric shapes, better performance

mask: Soft edges, gradients, complex patterns, more flexible

Use clip-path when: Simple shapes with hard edges

Use mask when: Need soft transitions or complex patterns

Browser Prefixes

Webkit prefix often needed for mask properties.

-webkit-mask-image: url(mask.png);
mask-image: url(mask.png);

Note: Include -webkit- prefix for better browser support
