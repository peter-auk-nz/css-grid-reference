What are CSS Values and Sizing?

CSS offers many ways to define sizes and measurements. Choosing the right unit determines how elements respond to different screen sizes and user settings. Understanding these units is essential for creating responsive, accessible designs.

Key Concept: Absolute units stay fixed, relative units adapt to context.

Absolute Units

Fixed measurements that don't change.

width: 300px;
font-size: 16pt;

px (pixels): Most common, 1px = 1 dot on screen

pt (points): 1pt = 1/72 inch, mainly for print

cm, mm, in: Physical measurements, rarely used for screens

Use: Borders, fine details, when you need exact pixel control

Relative Units - em

Relative to the parent element's font size.

padding: 1em; /* equals parent's font-size */
font-size: 1.5em; /* 1.5x parent's font-size */

Compounds: em values multiply through nested elements

Example: If parent is 16px, 1em = 16px, 2em = 32px

Use: Spacing around text, component-based scaling

Relative Units - rem

Relative to the root element's font size (usually html).

font-size: 1.5rem; /* 1.5x root font-size */
margin: 2rem; /* 2x root font-size */

Doesn't compound: Always relative to root, not parent

Default root: Usually 16px in browsers

Best practice: Use rem for most sizing for consistency

Viewport Units

Relative to the browser viewport (visible area).

width: 50vw; /* 50% of viewport width */
height: 100vh; /* 100% of viewport height */

vw: 1vw = 1% of viewport width

vh: 1vh = 1% of viewport height

vmin: Smaller of vw or vh

vmax: Larger of vw or vh

Use: Full-screen sections, hero images, responsive typography

Percentages

Relative to the parent element.

width: 50%; /* 50% of parent's width */
padding: 10%; /* 10% of parent's width */

Width/height: Relative to parent's dimension

Padding/margin: Always relative to parent's width (even for vertical)

Use: Flexible layouts, responsive containers

calc()

Performs calculations to mix different units.

width: calc(100% - 50px);
margin: calc(1rem + 10px);

Operators: + - * / (spaces required around + and -)

Mix units: Combine px, %, rem, vw, etc.

Use: Complex layouts, precise spacing, responsive math

min(), max(), clamp()

Modern functions for responsive sizing.

width: min(500px, 100%); /* whichever is smaller */
font-size: clamp(1rem, 2.5vw, 2rem); /* min, preferred, max */

min(): Use smallest value

max(): Use largest value

clamp(): Value between min and max, scales with middle value

Use: Fluid typography, responsive sizing without media queries
